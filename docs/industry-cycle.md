# 行业周期位置（v0.9）

为什么需要：白酒 / 煤炭 / 航空等强周期行业，单股财务可能很好，但若行业景气见顶，**估值回归压力大**——即使公司本身没问题。

## 数据源

**同花顺行业指数 5 年收盘价分位**（PE 数据接口在 akshare 损坏，用价格分位作 proxy）。

## 标签

- **高位** ≥ 80% → "核心资产→价值陷阱"风险
- **中位偏高** 50-80%
- **中位** 20-50%
- **底部** ≤ 20% → 周期反转机会，但需核对 CFO + 负债率

## 与 ROE 分位结合（v0.11）

行业 ROE 横截面分位**叠加**行业价格分位：

- ROE 在行业 top 20% + 行业指数高位 → ⚠ **周期顶部高 ROE 假象**
- ROE 在行业 top 20% + 行业指数非高位 → ✅ **个体阿尔法**

实例：陕煤当前 ROE 高，但煤炭行业指数 89% 高位 + ROE 行业排名也靠前 → 周期顶部，警惕业绩回归。

## 启用映射

仅对 30+ 周期性 / 消费 / 医药 / 金融行业启用（精确对齐同花顺 90 个行业名）。
其他行业静默跳过（白酒/科技/IT 服务等无明显周期性的不强制要求）。

详见 `stockwise/data/industry_cycle.py::_INDUSTRY_MAP`。
