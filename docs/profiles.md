# 6 类 Profile

stockwise 按行业**自动分发**到对应估值口径，避免"用一把尺量所有公司"。

## 概览

| Profile | 适用 | 安全边际口径 |
|---|---|---|
| **default** | 消费 / 制造 / 一般企业 | FCF Yield + Graham PE×PB + OE×12 + DCF 含增长 |
| **bank** | 银行 / 货币金融服务 | P/B + ROE÷PB 隐含回报 + 股息率 + 衰减 ROE 留存复利 |
| **insurance** | 保险 | P/B + 股息率 + 隐含回报 + 净利稳定性 |
| **growth** | 营收 5 年 CAGR ≥ 15-25%、研发驱动 | PEG（保守 g）+ (1/PE + g) + PS÷CAGR + DCF |
| **semi_growth** | CAGR 12-15% + 毛利 ≥ 30% | 估值同 default，维度门槛放宽 |
| **cyclical** | 钢铁/煤炭/有色/石油/航运/化工/水泥 | Shiller PE（10 年均值 EPS）+ P/B + 高股息率 + 周期顶部预警 |

## 识别规则

- **金融业**：行业名含「银行」「保险」「货币金融服务」→ bank / insurance（优先级最高）
- **周期股**：行业名含 `_CYCLICAL_INDUSTRY_KEYS` 关键词 → cyclical
- **成长股 A**：研发型行业 + 5y 营收 CAGR ≥ 15% → growth
- **成长股 B**：纯财务特征 CAGR ≥ 25% + 不在排除类 → growth
- **半成长 (v0.8)**：CAGR 12-15% + 5y 毛利 ≥ 30% + 不在排除类 → semi_growth
- 否则 → default

## 维度门槛对照

| 维度 | default | semi_growth | growth | bank | insurance |
|---|---|---|---|---|---|
| ROE 5y 门槛 | 15% | **12%** | 12% | 13% | 12% |
| 毛利门槛 | 40% | **30%** | 40% | n/a | n/a |
| ROA 门槛 | n/a | n/a | n/a | **1%** | n/a |
| 净利率门槛 | n/a | n/a | n/a | n/a | 8% |
| 净利 CAGR | 10% | **12%** | 25% | 8% | 8% |

## 评级 5 档

- **≥85 + 折价 ≥ 20%** → 值得长期持有
- **≥85 + 折价 ≥ 0%** → 优质合理估值
- **≥85 + 折价 < 0%** → 优质但偏贵
- **70-84 + 折价 ≥ 0%** → 质量好且估值合理
- **70-84 + 折价 < 0%** → 质量好但有瑕疵
- **< 70** → 未达伯克希尔标准
- **触发一票否决** → 避免
